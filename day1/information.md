# Day 1: AI Chatbot Integration and API Configuration

## 1.Objective
The primary goal of this task was to integrate an AI-powered assistant directly into the local development environment (code editor) using a secure, private API key. This setup provides immediate coding assistance, code explanation, and troubleshooting support directly within the workspace.

## 2. Tools and Technologies
- Code Editor: Visual Studio Code (VS Code)

- AI Model / Platform: Google Gemini / Google AI Studio (or specify your exact model, e.g., OpenAI, Claude)

- Authentication: Private API Key

## 3. Step-by-Step Implementation
## Step 3.1: Generating the Private API Key
1- Accessed the developer console or platform dashboard (e.g., Google AI Studio).

2- Navigated to the API management section and generated a unique, private API key.

## Step 3.2: Code Editor Configuration
1- Opened the code editor and installed the relevant AI assistant extension (e.g., Gemini Code Assist, Continue, or Codeium).

2- Accessed the extension settings menu within the editor.

3- Pasted the private API key into the designated authentication field to link the local editor to the remote AI model.

## Step 3.3: Testing and Verification
1- Initialized a chat session directly inside the editor's sidebar panel.

2- Submitted sample programming queries (e.g., asking for code explanations or debugging steps) to verify successful data transmission and response retrieval.

3- Confirmed that the model successfully processed the prompts and returned accurate, real-time code suggestions.

## 4. Key Takeaways & Learning Outcomes
- API Management: Gained practical experience in safely provisioning and handling private credentials.

- Workflow Optimization: Learned how integrating AI tools directly into an IDE reduces context-switching and speeds up code writing.
